#### red-packet-photo

red-packet-photo is imitation WeChat fuzzy picture , pay money to see.

red-packet-photo是仿微信模糊的图片，付费后才能看。

2016年春节微信朋友圈被红包图片刷屏，在这里给实现下红包图片。
